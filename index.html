1
Motor Assessment

Let's identify your motor type first:

Question 1: Can you see the motor's internal components?
A) Yes - I can see magnets on the inside wall (outrunner)
B) Yes - I can see a shaft going through the middle with windings around it (inrunner)
C) No - It's sealed or I'm not sure
Question 2: How many wires does the motor have?
A) 3 wires (most likely BLDC)
B) 2 wires (likely brushed DC)
C) More than 3 wires
Question 3: What's the approximate size?
A) Small (under 50mm diameter)
B) Medium (50-100mm)
C) Large (over 100mm)

Assessment Result:

This is most likely a BLDC outrunner motor - very common in small applications like fans, drones, or toys.

2
Safety First - Current Limiting
⚠️ Safety Critical Information

Proper current limiting prevents motor and ESC damage!

Safe Starting Assumptions:

  • Battery Voltage: 18V (as mentioned)
  • Safe Starting Current: 10-15A maximum
  • FSESC 6.7 Pro Capacity: Up to 45A, but salvaged motors may have unknown specifications

Set these in your mind:

  • Start with 10A current limit
  • Use 20A as absolute maximum for testing
  • Always test with motor unloaded first
3
Hardware Setup

Wiring Connections:

1. Motor to ESC:

  • Connect the 3 motor wires to any 3 of the motor output terminals (order doesn't matter initially)
  • The motor will spin the wrong direction if wires are in wrong order - easily fixed later

2. Battery to ESC:

  • Red wire to positive (+)
  • Black wire to negative (-)
  • Important: Ensure correct polarity!

3. USB to Computer:

  • Connect micro USB cable to the FSESC
  • Windows should install drivers automatically
4
Installing VESC Tool

Download and Install:

  • Go to vesc-project.com
  • Download VESC Tool for Windows
  • Install with default settings
  • Launch the program
Tip: If you have antivirus software, you may need to allow the installation as VESC Tool is sometimes flagged as suspicious.
5
Connecting to Your FSESC

Step-by-step Connection:

  • Open VESC Tool
  • Click "Connect" button (top left)
  • Select your connection type:
    • Usually "USB" for direct connection
    • Sometimes "UART" if USB doesn't work
  • If multiple ports appear, try each one
  • Success indicator: Connection status turns green and shows firmware version

If connection fails:

  • Check USB cable
  • Try different USB port
  • Ensure drivers installed
  • Check if device appears in Device Manager
6
Basic Motor Detection

Auto Detection Process:

  • Click "Wizard" in left sidebar
  • Select "Setup Motor Parameters"
  • Follow these steps carefully:

Motor Type Selection:

  • Choose "BLDC" (not brushed)
  • Select "Sensored" only if you see Hall sensor wires
  • Most likely choose "Sensorless"

Number of Poles:

  • Common values: 14, 22, or 28 poles
  • If unsure, start with 14 (very common for small motors)
7
Motor Configuration Wizard

Detailed Wizard Steps:

Step 1: Motor Parameters

  • Motor Type: BLDC, Sensorless
  • Number of Poles: Start with 14
  • Gear Ratio: 1:1 (unless you know otherwise)

Step 2: Battery Settings

  • Battery Cells: 4-5 cells (for 18V system)
  • Battery Type: Li-Ion or LiPo (most common)
  • Cut-off Voltage: 3.0V per cell minimum

Step 3: Current Limits (CRITICAL)

  • Motor Current Max: 10A (start here!)
  • Battery Current Max: 10A
  • Absolute Maximums: Set to 20A for safety

Step 4: Voltage Limits

  • Input Voltage Max: 20V (slightly above your 18V battery)
  • This protects against voltage spikes
8
Motor Detection Process

Running Motor Detection:

  • Ensure motor is free to spin (no load)
  • Remove propeller or load for safety
  • In Wizard, click "Detect Motor Parameters"
  • The motor will make noise and spin briefly

If it doesn't work:

  • Try swapping any two motor wires
  • Check connections
  • Try different pole count (14 → 22 → 28)
9
Testing Motor Direction

Checking Rotation:

  • After successful detection, go to "Controls" tab
  • Set "Control Type" to "PPM" (for RC control) or "Nunchuk" for testing
  • Click "Apply" and "Write"
  • Test carefully:
    • Use a small throttle input
    • Motor should spin smoothly
    • If spinning wrong direction, stop and proceed to next step

Fixing Direction:

  • Go to "Configuration" → "General"
  • Find "Motor Direction"
  • Change from "Forward" to "Reverse" (or vice versa)
  • OR swap any two motor wires
  • Test again
10
Fine-tuning Parameters

Optimizing Performance:

  • Go to "Configuration" → "Motor"
  • Important Settings:
    • Motor Current Max: Keep at 10A initially
    • Battery Current Max: 10A
    • RPM Max: Set to reasonable limit (start with 50,000 RPM)

Sensorless Settings:

  • Switching Frequency: 25,000 Hz (default)
  • Minimum ERPM: 150 (good for small motors)
11
Testing Under Load

Gradual Testing Process:

  • Start unloaded: Apply small throttle, listen for smooth operation
  • Increase gradually: Work up to 50% throttle
  • Check temperature: Feel motor after 1 minute - should be warm, not hot
  • Listen for noise: Smooth hum is good, grinding/buzzing is bad

If Motor is Rough or Noisy:

  • Go back to Wizard and try different pole count
  • Check "Motor Detection" again
  • Ensure proper wire connections
12
Saving Your Configuration
⚠️ Important - Save Your Work:
  • Once everything works smoothly:
  • Click "Configuration" → "Save Configuration"
  • Click "Write App" and "Write Motor"
  • This saves settings to the ESC memory
13
Troubleshooting Common Issues

Problem: Motor won't spin at all

Solutions:

  • Check all connections
  • Try swapping motor wire positions
  • Verify battery voltage (should be 16-18V)
  • Run motor detection again

Problem: Motor spins but roughly/vibrates

Solutions:

  • Wrong pole count - try 14, 22, or 28
  • Motor wires in wrong order - swap any two
  • Current too high - reduce to 5A temporarily

Problem: ESC gets very hot

Solutions:

  • Reduce current limits immediately
  • Check for short circuits
  • Ensure proper ventilation
14
Advanced Safety Settings

Recommended Safety Limits:

  • Go to "Configuration" → "General"
  • Set these safety values:
    • Battery Current Max: 15A
    • Motor Current Max: 15A
    • Temperature Limit: 80°C
    • Input Voltage Cutoff: 15V minimum
15
Final Testing Checklist

Before Full Operation:

  • Motor spins smoothly unloaded
  • Direction is correct
  • No unusual noises or vibrations
  • ESC doesn't overheat (5 minutes test)
  • Configuration is saved
  • Current limits are appropriate for motor size

Quick Reference: Safe Starting Values

Parameter Safe Starting Value Maximum for Testing
Motor Current 10A 15A
Battery Current 10A 15A
Max RPM 50,000 100,000
Battery Voltage 18V system 20V max
Pole Count 14 Try 22, 28 if needed

Emergency Stop Procedure

If anything goes wrong:

  1. Immediately disconnect battery
  2. Do not touch hot components
  3. Wait for components to cool before troubleshooting
  4. Check connections before reconnecting

🎉 Congratulations!

You should now have a working motor controller setup. Start with conservative settings and gradually increase performance as you verify the motor can handle it safely.
